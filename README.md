# SpacedGrid-WPF
An extension of WPF's Grid class with added RowSpacing and ColumnSpacing properties.

NuGet: https://www.nuget.org/packages/SpacedGrid-WPF/

Avalonia version: https://github.com/Nickelony/AvaloniaSpacedGrid/
